# 描述
统计字符串中整数的个数，如abc123egf，其中整数为1个，字符串中无整数时，返回0。


# 要求 
* 运行时间限制:	无限制
* 内存限制:	128 MByte

# 格式
* 输入:	输入字符串
* 输出:	输出字符串中整数的个数
* 样例输入:	a2sdkj321sjk
* 样例输出:	2

# 知识点
* 正则表达式，see refer [python regualr](https://docs.python.org/3.5/library/re.html)
* 整数：由自然数(正整数、零)和负整数组合，像-3,-2,-1,0,1,2,3,10等这样的数，不包括小数、分数。
